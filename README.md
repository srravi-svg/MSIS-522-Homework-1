# COVID-19 Mortality Risk Prediction

**MSIS 522 — Homework 1: The Complete Data Science Workflow**

## Overview

This project predicts COVID-19 patient mortality using machine learning. We train and compare five models — Logistic Regression, Decision Tree, Random Forest, LightGBM, and a Neural Network (Keras MLP) — on a balanced sample of 10,000 patients from Mexico's COVID-19 surveillance data. The best model is explained using SHAP, and the entire analysis is deployed as an interactive Streamlit app.

## How to Reproduce

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the analysis notebook

Open `covid19_analysis.ipynb` in Jupyter or Google Colab and run all cells. This will:
- Perform exploratory data analysis (Part 1)
- Train and tune all five models (Part 2)
- Run SHAP explainability analysis (Part 3)
- Save all models to `models/` and all plots to `plots/`

### 3. Run the Streamlit app locally

```bash
streamlit run streamlit_app.py
```

### 4. Deploy to Streamlit Community Cloud

1. Push this repository to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub repo and select `streamlit_app.py`
4. Deploy

## Dataset

The dataset comes from Mexico's national COVID-19 epidemiological surveillance system. It contains 16 features (15 binary + 1 continuous) and a binary target (`DEATH`). After balanced sampling: 5,000 died + 5,000 survived = 10,000 total patients.

**Features:** SEX, HOSPITALIZED, PNEUMONIA, AGE, PREGNANT, DIABETES, COPD, ASTHMA, IMMUNOSUPPRESSION, HYPERTENSION, OTHER_DISEASE, CARDIOVASCULAR, OBESITY, RENAL_CHRONIC, TOBACCO, COVID_POSITIVE

Top predictive features (via SHAP): **Hospitalization status**, **Age**, and **Pneumonia**.
